# 🚀 Deploy Rápido a Vercel

## Opción 1: Botón de Deploy (Automático)

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/tu-usuario/bfrezko-replica)

## Opción 2: CLI (Más Control)

```bash
# 1. Instalar Vercel CLI
npm i -g vercel

# 2. Login (abre navegador)
vercel login

# 3. Deploy interactivo
vercel

# 4. Deploy producción
vercel --prod
```

## Opción 3: Git Integration (Recomendado)

1. Push código a GitHub
2. Conectar repositorio en vercel.com
3. Auto-deploy en cada push a `main`
